#### 기능

- 상품
1. 상품 목록 조회 
   * url = GET "/list"

2. 상품 단건 조회
    * url = GET "/detail"

3. 상품 생성
    * url = PUT "/detail"

4. 상품 수정
    * url = POST "/detail"

5. 상품 단건 삭제
    * url = DELETE "/detail"
